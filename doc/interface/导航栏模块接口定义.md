### 导航栏模块接口定义

</br>

#### `疫苗接种` 主界面

* elementId: `<router-view>`

* VaccineHome.vue

* fetch:

  * url: `/vaccineHome/worldMapVaccineDataMsg`
  * 变量：无
  * type: `GET`
  * data:

  ```json
  {
      vaccined: {				// 累计接种
          {
          	name:		// 国家名称：String（中文）
          	value:		// 接种人数：Integer
      	},
      	......
      },
      coverage: {				// 覆盖率
       	{
          	name:
          	value:		// 覆盖率：FLOAT∈[0,1]，保留三位小数
      	},
      	......
      }
  }
  ```

</br>
