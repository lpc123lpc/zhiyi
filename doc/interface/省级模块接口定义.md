### 省级模块接口定义

</br>

#### `感染情况` 省级界面

* elementId: `<router-view>`

* InfectProvinceDetail.vue

* AJAX

  * url: `/infectDetail/provinceMapInfectionDataMsg/<province>`
  * 变量：`<province>` （当前所在省）
  * type: `GET`
  * data:

  ```json
  {
      nowConfirm: {				// 现有确诊
          {
          	name:		// 省名称：String（中文）
          	value:		// 确诊人数：Integer
      	},
      	......
      },
      totalConfirm: {	...... },	// 累计确诊
      cured: { ...... },			// 累计治愈
      dead: { ...... }			// 累计死亡
  }
  ```
